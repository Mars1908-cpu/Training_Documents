	- Containerization :::::		
	
		- Is a process used to package the application along with its dependencies.
		
		- All the Micro-Service based application service are Containerized and deployed using Kubernetes to ensure High availability.
		
	- Virtual Machines (vs) Containers :::
	
	What is Virtual Machine?
		
	 - VMs are the Hardware Level Virtualization
	 - VMs are created using Hypervisor 
	 - VMs are used to execute the Operating System
	 - VMs can continuously up and running even if there is no active application/task
	 - VMs consume more space and time to start the application		
		
	What is Containers?
		
	 - Containers are the Operating System (OS) Level Virtualization
	 - Containers are created using Container Engine(Docker) 
	 - Containers are used to execute the Applications/Tasks not the Operating System.
	 - Containers will immediately go to EXIT State if there is no active Applications/Tasks
	 - Containers consume less space and time to start the Applications/Tasks
	 - Containers run in its own dedicated and completely isolated Address-Space.
	 
		--> Namespace & Control Groups 	are the core properties of the kernel(Base OS)
		
		--> Using Containers we can reduce the number of VMs. But, we cannot completely eliminate VMs.

	- Containerization Concepts & Terminologies 	


		- Containerization 		# Is a process used to package the application along with its dependencies.
		
		- Container Engine 		# Used to create and Manage the Container Images and Containers
		
		- Container Image 		# Is a static file, that defines the properties & the dependencies of the Container
								# It is non-executable 
								# It composed of various layers of the Instruction created using Dockerfile
								
		- Container				# Is the Executable unit of Container Image
		
		- Container Registry	# Used to Version Control the Container Images
								# DockerHub is the default Container Registry for Docker Container Engine 
									Eg.: DockerHub 
									https://hub.docker.com/
								
		- Container Repository 	# Is the Subset of the Container Registry
